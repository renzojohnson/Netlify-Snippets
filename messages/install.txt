  Netlify Snippets for Sublime Text
  ==================================

  125 curated snippets for Netlify development.
  Built from Netlify docs (March 2026).

  Quick Start:
    1. Open any JS/TS/TOML/HTML file
    2. Type  nt:  and browse the autocomplete menu
    3. Press Tab to expand and jump between placeholders
